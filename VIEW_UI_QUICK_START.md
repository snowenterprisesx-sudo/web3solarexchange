# ⚡ Quick Start: View Your UI in Kiro (30 Seconds)

## **3 Simple Steps**

### **Step 1: Open Terminal in Kiro**
```
Press: Ctrl + ~
Or: Click "Terminal" tab at bottom
```

### **Step 2: Run Dev Server**
```bash
npm run dev
```

### **Step 3: Click the Link**
```
Look for: Local: http://localhost:5173
Click it!
```

## **Troubleshooting**

| Problem | Solution |
|---------|----------|
| Port 5173 in use | `lsof -ti:5173 \| xargs kill -9` then `npm run dev` |
| Blank page | Press F12, check Console for errors |
| Can't find npm | Install Node.js from nodejs.org |
