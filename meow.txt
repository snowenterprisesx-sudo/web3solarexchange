# 🧠 SYSTEM OVERVIEW

This is a PRIVATE SXP BRIDGE system using:

* Mint/Burn wrapped token model (wSXP)
* NO double funding requirement
* Backend verification layer
* Wallet-based transaction signing

---

# 🔷 CONTRACTS (Solidity)

Create:

1. WSXP.sol

* ERC20 token
* mint(address, amount)
* burn(address, amount)

2. Vault.sol

* Holds reserve funds
* deposit()
* release(address, amount)

Use OpenZeppelin libraries.

# ⚙️ BACKEND (Node + Express)

Install:

* express
* ethers
* cors
* dotenv
* jsonwebtoken
* bcrypt

## Core Features:

1. RPC connections (ETH + BNB)
2. Deposit listener
3. Mint wSXP when deposit detected
4. Burn wSXP when user withdraws
5. Release funds from vault

## API ENDPOINTS

POST /api/bridge/deposit
