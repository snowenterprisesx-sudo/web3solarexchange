## **🆘 Troubleshooting**

| Problem | Solution |
|---------|----------|
| Port 5173 in use | `lsof -ti:5173 \| xargs kill -9` then `npm run dev` |
| Blank page | Press F12, check Console for errors |
| npm not found | Install Node.js from nodejs.org |
| Animations slow | Close other browser tabs |
