## **Troubleshooting**

### **"No such file or directory"**
- You're in the wrong folder
- Check your project path
- Navigate to correct location

### **"package.json not found"**
- You're not in the project root
- Run `dir` to see what's in current folder
- Navigate to correct location

### **"npm: command not found"**
- Node.js not installed
- Install from https://nodejs.org
- Restart your computer

### **"Port 5173 already in use"**
- Another app is using that port
- Close other terminals running `npm run dev`
- Or use: `npm run dev -- --port 5174`

### **Blank page or errors**
- Wait 30 seconds for build
- Refresh browser (F5)
- Check Command Prompt for errors
